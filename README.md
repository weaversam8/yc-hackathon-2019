# yc-hackathon-2019

YC Hackathon 2019.

## Setup instructions

-   Run python 3
-   Install `nltk` through `pip`
-   Install `PyYAML` through `pip`
-   Run `python` and enter in the interactive prompt the following commands:
    -   `import nltk`
    -   `nltk.download('punkt')`
    -   `nltk.download('stopwords')`
